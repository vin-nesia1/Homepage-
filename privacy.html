Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana VIN NESIA mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda menggunakan situs web dan tools kami. Kami sangat menghargai privasi Anda.

1. Informasi yang Kami Kumpulkan

VIN NESIA tidak mengumpulkan informasi pribadi yang dapat diidentifikasi secara langsung (seperti nama, alamat email, atau nomor telepon) dari pengguna tools kami. Tools kami beroperasi sepenuhnya di sisi klien (browser Anda), yang berarti data yang Anda masukkan untuk diproses tidak pernah dikirim atau disimpan di server kami.

Kami mungkin mengumpulkan informasi non-pribadi secara otomatis melalui Google Analytics (ID Pelacakan: G-JXL70YP8XN) untuk tujuan analisis lalu lintas situs web. Informasi ini dapat mencakup:

  • Jenis browser dan perangkat yang Anda gunakan.
  • Halaman yang Anda kunjungi di situs kami.
  • Waktu yang dihabiskan di setiap halaman.
  • URL rujukan (dari mana Anda datang ke situs kami).
  • Informasi geografis umum (negara atau kota, bukan lokasi spesifik).
Data ini digunakan untuk memahami bagaimana pengguna berinteraksi dengan situs kami, sehingga kami dapat meningkatkan pengalaman pengguna dan fungsionalitas tools. Data ini dianonimkan dan tidak dapat digunakan untuk mengidentifikasi Anda secara pribadi.

2. Penggunaan Informasi

Informasi non-pribadi yang kami kumpulkan digunakan semata-mata untuk:

  • Menganalisis tren penggunaan situs.
  • Meningkatkan desain dan fungsionalitas situs.
  • Memahami preferensi pengguna untuk pengembangan tools di masa mendatang.
Kami tidak akan menjual, menyewakan, atau membagikan informasi non-pribadi ini kepada pihak ketiga untuk tujuan pemasaran.

3. Keamanan Data

Karena tools kami beroperasi di sisi klien, data yang Anda masukkan tidak pernah meninggalkan browser Anda dan tidak disimpan di server kami. Ini secara inheren meningkatkan keamanan data Anda. Kami juga menggunakan protokol HTTPS untuk memastikan komunikasi yang aman antara browser Anda dan situs web kami.

4. Tautan ke Situs Pihak Ketiga

Situs web kami mungkin berisi tautan ke situs web pihak ketiga. Kebijakan Privasi ini hanya berlaku untuk VIN NESIA. Kami tidak bertanggung jawab atas praktik privasi atau konten situs web pihak ketiga tersebut. Kami mendorong Anda untuk membaca kebijakan privasi setiap situs web yang Anda kunjungi.

5. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diposting di halaman ini dengan tanggal "Pembaruan Terakhir" yang direvisi. Anda disarankan untuk meninjau Kebijakan Privasi ini secara berkala untuk setiap perubahan.
